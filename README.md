# AI Coding Skills Wiki

`VitePress` 기반으로 다양한 AI coding `skills`를 한국어로 정리하고, `GitHub Pages`로 배포하는 정적 문서 사이트다.

- 저장소: `https://github.com/seihwanMoon/skillrnd`

## 목표

- 여러 agent 생태계의 `skill`을 한 곳에서 탐색
- `설명`, `분류`, `Mermaid`, `원문 링크`, `활용 레시피`를 일관된 형식으로 제공
- 이후 `Codex` 기반 자동 수집기와 문서 생성기로 확장

## 로컬 실행

```bash
npm install
npm run docs:dev
```

## 빌드

```bash
npm run docs:build
```
