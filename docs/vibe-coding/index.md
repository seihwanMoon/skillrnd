# Vibe Coding

이 섹션은 `생성 -> 실행 -> 검증 -> 수정 -> 문서화`의 루프를 중심으로 AI coding workflow를 정리한다.

## 핵심 질문

- 어떤 도구 조합이 가장 빠른가
- 생성된 코드를 어떻게 검증할 것인가
- 문서와 테스트를 같은 루프 안에 넣을 수 있는가
