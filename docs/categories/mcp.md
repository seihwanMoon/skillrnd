# MCP 카테고리

도구 등록, 리소스 연결, ChatGPT Apps SDK, 브리지 연동처럼 `MCP` 또는 도구 인터페이스 설계에 초점을 둔 skill을 다룬다.

## 빠른 선택 가이드

- ChatGPT App과 위젯 UI, MCP 서버를 함께 설계하려면 [chatgpt-apps](/skills/chatgpt-apps)
- 외부 또는 공식 skill을 실제 환경에 설치하려면 [skill-installer](/skills/skill-installer)

## 관찰 포인트

- 도구 정의와 메타데이터 관리 방식
- UI와 도구 브리지 연결 방식
- 설치와 배포 흐름의 단순성
- 특정 플랫폼 의존성 여부

## 대표 문서

- [chatgpt-apps](/skills/chatgpt-apps)
- [skill-installer](/skills/skill-installer)

## 추천 조합

1. `skill-installer`로 필요한 skill을 확보
2. `chatgpt-apps`로 실제 앱 구조와 도구 브리지를 설계
3. 관련 recipe와 references를 통해 문서화까지 연결
