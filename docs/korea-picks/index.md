# Korea Picks

이 섹션은 한국어 사용자와 한국 실무 환경에 맞는 도구, skill, MCP, 데이터 소스를 정리한다.

## 왜 따로 다루는가

- 글로벌 생태계와 한국 실무의 요구가 다르다.
- 법령, 공공데이터, 금융, 제조 환경은 한국 특화 도구 가치가 크다.
- 한국어 문서화와 현업 적용성 기준이 따로 필요하다.

## 우선 분류

- 법령/규제
- 금융/공시
- 공공데이터
- 제조/산업
- 한국어 업무 자동화

## 선정 기준

- 한국 실무 맥락에서 바로 쓸 수 있는가
- 문서와 설치 흐름이 어느 정도 검증 가능한가
- 한국어 사용자에게 탐색 가치가 높은가

## 운영 방식

- 초기에는 수동 큐레이션
- 이후에는 별도 `Korea Picks Board`로 확장
- 가능하면 설치, 검증, 활용 예시를 함께 기록
