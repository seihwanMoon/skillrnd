# playwright

## 한줄 요약

실제 브라우저를 조작해 UI 흐름을 검증하고 스냅샷, 스크린샷, 데이터 추출을 수행하는 자동화 skill이다.

## 분류

- Agent: `Codex`
- Purpose: `testing`
- Shape: `single skill`

## 언제 쓰는가

- 실제 브라우저 동작이 필요한 테스트를 작성할 때
- 폼 입력, 로그인, 스크린샷 자동화가 필요할 때
- UI 디버깅과 흐름 재현이 필요할 때

## 입력과 출력

- 입력: URL, 브라우저 상호작용 시나리오
- 출력: 스냅샷, 스크린샷, 동작 로그, 추출 데이터

## Mermaid

```mermaid
flowchart LR
  A[브라우저 실행] --> B[페이지 이동]
  B --> C[상호작용 수행]
  C --> D[검증 또는 캡처]
  D --> E[결과 기록]
```

## 장점

- 실제 UI 흐름 검증에 직접적이다.
- 테스트와 디버깅에 모두 활용 가능하다.

## 한계

- 환경 차이와 타이밍 이슈에 민감할 수 있다.
- 사이트 상태에 따라 테스트가 흔들릴 수 있다.
