# GitHub Study Methodology

이 섹션은 `poket` 파이프라인이 수집한 영문 기술 자료를 한국어 해설 문서로 재구성해 누적 출판합니다.

## 운영 원칙

- 원문 링크를 보존합니다.
- 장문 원문 복제 대신 해설과 요약 중심으로 작성합니다.
- 생성 문서는 `generated/` 아래에 entry 단위로 축적합니다.
- 원문 발췌 페이지는 탐색 메뉴에서 숨기고 직접 링크로만 접근합니다.
