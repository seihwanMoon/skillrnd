# 랭킹

이 섹션은 단순 인기 목록이 아니라, 공개 지표와 수동 큐레이션을 결합한 `추천 보드`를 목표로 한다.

## 현재 원칙

- `조회수` 단독 기준은 사용하지 않는다.
- 공개적으로 확인 가능한 지표를 우선 쓴다.
- `인기`와 `품질/적합도`를 분리해 본다.

## 예정 보드

- 인기 상위
- 최근 급상승
- 초심자 추천
- 한국 실무 추천
- 검증 완료
