# 문서 생성 레시피

## 목적

분산된 `skill` 설명을 한국어 문서 허브로 정리한다.

## 입력 체크리스트

- 대상 skill 또는 저장소 목록
- 문서 템플릿
- 카테고리 기준
- 원문 링크와 확인 가능한 근거

## 권장 흐름

1. 대상 skill 선정
2. 원문 `SKILL.md` 확인
3. 공통 템플릿에 맞게 한국어 문서 작성
4. Mermaid로 구조 흐름 추가
5. 카테고리와 관련 문서 링크 연결

## 단계별 실행

### 1단계

문서화 우선순위를 정하고 로컬 skill과 외부 저장소를 구분한다.

### 2단계

원문 `SKILL.md`, README, 설치 흐름을 확인해 사실 기반 메모를 만든다.

### 3단계

표준 템플릿에 맞춰 `한줄 요약`, `언제 쓰는가`, `입력과 출력`, `핵심 구조`, `Mermaid`, `장점`, `한계`, `링크`를 채운다.

### 4단계

카테고리, 레퍼런스, 레시피 허브에 링크를 연결해 탐색성을 높인다.

## 체크포인트

- 사실과 추정 구분
- 원문 링크 유지
- 카테고리 일관성 유지

## 결과물

- 개별 skill 문서
- 비교 가능한 카테고리 문서
- 관련 레퍼런스 링크

## Mermaid

```mermaid
flowchart LR
  A[대상 선정] --> B[원문 확인]
  B --> C[한국어 템플릿 작성]
  C --> D[Mermaid 추가]
  D --> E[카테고리와 레퍼런스 연결]
```
