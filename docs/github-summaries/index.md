# GitHub Summaries

공개 GitHub 저장소를 초보자도 읽을 수 있는 한국어 매뉴얼형 문서로 정리한 아카이브입니다.

## 이 섹션에서 다루는 것

- 저장소 목적 요약
- 폴더 구조와 실행 흐름 정리
- 빠른 시작, 주요 파일, Mermaid 다이어그램

## 운영 원칙

- 공개 repo 루트 URL만 1차 지원
- 구조와 실행 흐름을 우선 설명
- 불확실한 내용은 `추정`, `확인 필요`로 표시
